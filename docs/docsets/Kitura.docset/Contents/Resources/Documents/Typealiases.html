The following typealiases are available globally.

  • RouterHandler

    The definition of the closure type that is used by the Router class when routing HTTP requests to closure.

    Parameter

    Parameter request: The RouterRequest object that is used to work with the incoming request.

    Parameter

    Parameter response: The RouterResponse object used to send responses to the HTTP request.

    Parameter

    Parameter next: The closure to invoke to cause the router to inspect the path in the list of paths.

    Swift
