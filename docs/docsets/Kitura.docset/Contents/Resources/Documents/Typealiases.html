Type Aliases

The following type aliases are available globally.

  • RouterHandler

    The definition of the closure type that is used by the Router class when routing HTTP requests to closure.

    Declaration

    Swift

    public typealias RouterHandler = (RouterRequest, RouterResponse, @escaping () -> Void) throws -> Void

    Parameters

    request

    The RouterRequest object that is used to work with the incoming request.
    response

    The RouterResponse object used to send responses to the HTTP request.
    next

    The closure to invoke to cause the router to inspect the path in the list of paths.
